# WorkTrace AI Desktop

Initial Tauri v2, React, TypeScript, and Tailwind desktop shell for WorkTrace AI.

This app currently shows foundation status only. It does not implement recording, capture workers, sidecar lifecycle, SQLite UI, OCR, embeddings, local models, or AI reports.

## Recommended IDE Setup

- [VS Code](https://code.visualstudio.com/) + [Tauri](https://marketplace.visualstudio.com/items?itemName=tauri-apps.tauri-vscode) + [rust-analyzer](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer)
